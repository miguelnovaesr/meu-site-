Guia Básico de CSS Grid e Flexbox

CSS Grid Layout

Algumas das principais propriedades do CSS Grid (baseado no guia da Origamid):

  • display: grid; — transforma o elemento em grid container. :contentReference[oaicite:0]{index=0}
  • grid-template-columns — define colunas do grid. :contentReference[oaicite:1]{index=1}
  • grid-template-rows — define linhas. :contentReference[oaicite:2]{index=2}
  • grid-template-areas — agrupa áreas nomeadas para posicionamento. :contentReference[oaicite:3]{index=3}
  • gap — define espaçamento entre itens (linhas e colunas). :contentReference[oaicite:4]{index=4}
  • grid-auto-flow — controla como itens implicítos são colocados (linhas ou colunas). :contentReference[oaicite:5]{index=5}
  • justify-content, align-content — alinham o conjunto de itens no container. :contentReference[oaicite:6]{index=6}
  • justify-items, align-items — alinham os itens dentro de suas células. :contentReference[oaicite:7]{index=7}
  • grid-column, grid-row — posicionam cada item em colunas/linhas específicas. :contentReference[oaicite:8]{index=8}
  • grid-area — atalho para definir início/fim de linha e coluna para um item. :contentReference[oaicite:9]{index=9}
  • justify-self, align-self — alinham individualmente cada item na sua célula. :contentReference[oaicite:10]{index=10}

CSS Flexbox

Principais propriedades do Flexbox (baseado no guia do CSS-Tricks):

  • display: flex; — torna o elemento um flex container. :contentReference[oaicite:11]{index=11}
  • flex-direction — define a direção dos itens (row, column, row-reverse, etc). :contentReference[oaicite:12]{index=12}
  • flex-wrap — controla se os itens quebram a linha (wrap, nowrap, wrap-reverse). :contentReference[oaicite:13]{index=13}
  • flex-flow — atalho para flex-direction + flex-wrap. :contentReference[oaicite:14]{index=14}
  • justify-content — alinha itens no eixo principal. :contentReference[oaicite:15]{index=15}
  • align-items — alinha itens no eixo transversal. :contentReference[oaicite:16]{index=16}
  • align-content — alinha linhas de itens quando há múltiplas linhas (caso use wrap). :contentReference[oaicite:17]{index=17}
  • flex-grow — define quanto um item pode crescer para preencher espaço extra. :contentReference[oaicite:18]{index=18}
  • flex-shrink — define se o item pode encolher se não houver espaço. :contentReference[oaicite:19]{index=19}
  • flex-basis — define tamanho base antes de distribuir o espaço livre. :contentReference[oaicite:20]{index=20}
  • flex — atalho para flex-grow, flex-shrink, e flex-basis. :contentReference[oaicite:21]{index=21}
  • order — reordena os itens dentro do container. :contentReference[oaicite:22]{index=22}
